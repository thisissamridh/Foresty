Fuzzer tab

The Fuzzer tab shows you the requests and responses performed when you fuzz sebuah string.
Memilih turut melihat penuh permintaan dan tanggapan. Anda juga dapat mencari string dalam fuzz hasil menggunakan tab Pencarian.

Hasil HTTP Fuzzer

Hasil harus secara manual dinilai untuk mengetahui jika ada kerentanan yang ditemukan.

Menu klik kanan

Right clicking on a row will bring up a menu which has the same options as the History tab. Plus providing an option to add messages from the fuzz results to the Sites Tree and History tab (the messages will be tagged FromFuzzer and initially show a fuzzer icon in the Sites Tree).
