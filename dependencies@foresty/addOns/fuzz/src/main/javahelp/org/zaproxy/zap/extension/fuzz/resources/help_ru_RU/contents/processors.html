Диалоговое окно процессоров полезной нагрузки

Это позволяет вам выбирать процессоры полезной нагрузки для использования с конкретными генераторами полезной нагрузки.

Встроенные процессоры полезной нагрузки включают:
  • Декодирование Base 64
  • Кодирование Base64
  • Расширить (до минимальной указанной длины)
  • JavaScript Escape (Экранирование)
  • JavaScript Unescape
  • Хэширование MD5
  • Постфикс Cтрока:
  • Префикс Строка:
  • Хэширование SHA1
  • SHA256 Хэш
  • SHA-512 Хэш
  • Отделка
  • Декодирование URL
  • Кодирование URL
Вы также можете написать пользовательские сценарии процессора полезной нагрузки — они могут выполнять любые манипуляции с полезной нагрузкой, которые вам нужны.

Надстройки также могут определять дополнительные процессоры полезной нагрузки.
