Диалог фаззера

Это позволяет вам выбрать фаззеры для использования, когда фаззинг запрос.

Местоположение FUZZ вкладка

Чтобы настроить фаззинг:
  • Выделите строку, которую вы хотите FUZZ в Tab Puzz Locations
  • Нажмите кнопку «Добавить ...», чтобы запустить диалог полезных нагрузок
  • Добавьте полезные нагрузки, которые вы хотите использовать
  • Нажмите кнопку «Процессоры ...», чтобы запустить диалоговое окно процессоров location (если требуется)
  • Нажмите на кнопку «Пуск», чтобы начать попадание
  • Результаты будут затем перечислены вФаззер вкладка выберите их, чтобы увидеть полные запросы и ответы.
Вы также можете искать строки в результатах FUZZ, используя вкладку «Поиск».

Нажмите на кнопку «Редактировать», чтобы редактировать сообщение, которое вы выбрали для незма. Обратите внимание, что это удалит все определенные вами местоположения FUZZ.

Вам нужно будет «сохранить» сообщение, прежде чем вы сможете определить новые местоположения FUZZ.

Вкладка Опции Эта вкладка позволяет настроить параметры, которые будут использоваться при фаззинге. См. Основные Параметры помощи для более подробной информации.

Следуйте перенаправленным

Указывает, следует ли нет Фаззеру следовать перенаправленным ответами (видимым только для HTTP-сообщений).

Вкладка процессоров сообщений

Процессоры сообщений могут получить доступ и изменять разпущенные сообщения, контролировать процесс напыления и взаимодействовать с zap ui.
Доступные процессоры сообщений зависят от типов путаного сообщения, это дополнение включает в себя процессоры HTTP-сообщений по умолчанию .

Add-ons также может определить дополнительные генераторы полезных нагрузок.
