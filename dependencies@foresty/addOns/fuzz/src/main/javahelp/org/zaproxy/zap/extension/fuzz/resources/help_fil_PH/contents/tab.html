Tab ng Fuzzer

Ang tab ng fuzzer ay nagpapakita sa iyo ng mga kahilingan at tugon na isinagawa kapag ikaw ay nag-fuzz ng string.
Ang pagpipili ng isang hilera ay nakakita ng buong mga kahilingan at tugon. Maaari ka ring maghanap ng mga string sa mga resulta ng fuzz gamit ang tab ng paghahanap.

Mga resulta ng Fuzzer ng HTTP

Ang mga resulta ay kailangang manu-manong tasahin upang malaman kung may natagpuan na anumang kahinaan.

Menu sa pag-right click

Right clicking on a row will bring up a menu which has the same options as the History tab. Plus providing an option to add messages from the fuzz results to the Sites Tree and History tab (the messages will be tagged FromFuzzer and initially show a fuzzer icon in the Sites Tree).
