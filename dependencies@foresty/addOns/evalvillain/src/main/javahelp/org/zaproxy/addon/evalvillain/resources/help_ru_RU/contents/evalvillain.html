Eval Villain

Это дополнение добавляет расширение Eval Villain в Firefox при запуске из ZAP.
Eval Villain — это веб-расширение для Firefox, которое перехватывает опасные функции, такие как eval, и предупреждает вас об их использовании.

Дополнительные сведения см. в записи блога ZAP Дополнение Eval Villain и Домашняя страница Eval Villain.

Нажмите кнопку Eval Villain на панели инструментов Firefox, чтобы начать использовать его.

Вы можете включать и отключать Eval Villain и другие расширения браузера на странице настроек Selenium.
