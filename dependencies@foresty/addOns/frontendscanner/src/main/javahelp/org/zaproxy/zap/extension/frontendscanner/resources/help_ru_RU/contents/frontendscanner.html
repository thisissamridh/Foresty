Внешний интерфейс Сканер

При установке и включении надстройка Front-End Scanner вмешивается во все HTTP-ответы, проходящие через ZAP, чтобы внедрить фрагмент кода JavaScript, содержащий пользовательскую логику, служебные функции и сценарии, которые пользователи ZAP хотят запускать в браузере, чтобы помочь им проверить фронт. -конечные приложения. Примечание: для полноценной работы надстройка автоматически удаляет заголовки CSP с веб-страниц.

Добавьте скрипт для запуска в браузере

Примечание. на данный момент в веб-страницы внедряются только "пассивные клиентские" сценарии.

Чтобы добавить пассивный сценарий, перейдите на панель «Сценарии» и щелкните правой кнопкой мыши «Пассивные правила на стороне клиента», затем «Новый сценарий...». Дайте вашему сценарию осмысленное имя, убедитесь, что вы используете шаблон «scan.js», и сохраните его.

Поскольку они будут выполняться в веб-браузере, сценарии необходимо писать на Javascript. По той же причине они могут использовать веб-API, предоставляемые используемым вами браузером.

Каждый пользовательский скрипт, написанный через интерфейс ZAP, завершается в функции внешнего интерфейса, получая объект Javascript (FrontEndScanner) в качестве параметра. Таким образом, скрипты имеют доступ к тому, что выставляет этот объект:

  • константы ZAP, помогающие сценариям создавать оповещения,
  • «почтовый ящик»: механизм «публикация-подписка», помогающий сценариям реагировать на события, происходящие в браузере (такие как взаимодействие с пользователем, доступ к хранилищу и т. д.),
  • вспомогательная функция для сообщения о результатах обратно в ZAP.
Все функции, содержащие пользовательские (пассивные) скрипты, будут автоматически выполняться в браузере.

Включение — выключение

Чтобы переключить надстройку, перейдите в «Инструменты», затем «Переключить внешний сканер» (обратите внимание, что по умолчанию надстройка отключена).

Примечание. На целевые веб-страницы будут внедряться только разрешенные скрипты. Чтобы включить (или отключить) определенные сценарии, щелкните их правой кнопкой мыши на панели «Сценарии», затем выберите «Включить сценарии» (или, соответственно, «Отключить сценарии»).
