Опции Экран кодирования/декодирования

На этом экране можно настроить параметры Кодировать/Декодировать/Хэшировать:

Base64 — кодировка

Позволяет пользователю выбрать, какой набор символов следует использовать для преобразования Base64.

Base64 — линии разрыва

Указывает, должна ли кодировка Base64 разбивать строки на 76 символов (стандарт MIME).
