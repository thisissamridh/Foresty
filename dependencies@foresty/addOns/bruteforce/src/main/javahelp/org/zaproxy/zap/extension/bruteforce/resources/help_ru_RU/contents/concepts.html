Принудительный просмотр

ZAP позволяет вам попытаться обнаружить каталоги и файлы с помощью принудительного просмотра.
Предоставляется набор файлов, который содержит большое количество имен файлов и каталогов.
ZAP пытается получить прямой доступ ко всем файлам и каталогам, перечисленным в выбранном файл напрямую, а не полагаться на поиск ссылок на них.

Принудительный просмотр настраивается с помощью Опций. Экран принудительного просмотра .

Эта функциональность основана на коде устаревшего проекта OWASP DirBuster.

Доступ через

     Вкладка «Принудительный просмотр»
     Вкладка "Сайты" Пункт контекстного меню «Атака/Принудительный просмотр сайта»
     Вкладка "Сайты" Пункт меню «Атака/Принудительный просмотр каталога»
     Вкладка "Сайты" Пункт меню «Атака/Принудительный просмотр каталога (и дочерних элементов)»
