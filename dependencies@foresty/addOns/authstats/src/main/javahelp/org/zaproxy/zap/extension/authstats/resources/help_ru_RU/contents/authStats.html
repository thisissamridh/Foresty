Статистика аутентификации

Эта надстройка записывает статистику, связанную с аутентификацией, для всех контекстов, входящих в область действия.
Статистика ведется для каждого сайта и охватывает:
stats.auth.компонент.state.noauthВ контексте нет метода аутентификации
stats.auth. компонент .state.nothtml Ответы не в формате HTML (которые вряд ли содержат индикаторы входа / выхода)
stats.auth. компонент .state.notsuccess Неудачные HTML-ответы (которые, скорее всего, не связаны с индикаторами входа / выхода)
stats.auth. компонент .state.noindicator Индикаторы входа / выхода не определены
stats.auth. компонент .state.loggedinandout Индикаторы входа и выхода присутствуют.
stats.auth. компонент .state.loggedin Присутствует только индикатор авторизации.
stats.auth. компонент .state.loggedout Присутствует только отключенный индикатор
stats.auth. компонент .state.unknown Индикаторы входа или выхода отсутствуют
Компонент может быть:
сканирование активным сканером
авторизация Аутентификация
фаззингФаззер
ВручнуюРучные запросы
Прокси-серверПрокси-запросы
паук Паук
idЦелочисленный идентификатор другого компонента, не указанного выше

статистику можно получить через ZAP API.
