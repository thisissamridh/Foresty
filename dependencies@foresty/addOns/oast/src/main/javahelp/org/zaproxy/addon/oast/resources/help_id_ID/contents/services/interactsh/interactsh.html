Interactsh

Interactsh is an open-source solution for out-of-band data extraction.

Features

  • DNS/HTTP/SMTP Interaction support
  • AES encryption with zero logging
  • Self hosted server support
  • Automatic ACME based Wildcard TLS w/ Auto Renewal

Requests made to Interactsh by the target web application can be viewed in the OAST Callbacks Tab in the ZAP GUI.
