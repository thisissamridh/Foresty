Платформа автоматизации - графический интерфейс

У Automation Framework есть графический интерфейс, который находится в процессе разработки.

Вкладка "Автоматизация"

Эта вкладка позволяет создавать, загружать, редактировать и запускать задания автоматизации. Он имеет 2 дополнительных вкладки.

Панель инструментов содержит следующие кнопки:

  • Новый план ... - запускает диалоговое окно "Новый план".
  • Загрузить план ... - позволяет загрузить план из файла yaml.
  • Сохранить план ... - сохраняет текущий план в файл yaml.
  • Выполнить план ... - запускает текущий план.
  • Добавить задание ... - запускает диалоговое окно Добавить задание, чтобы добавить задание в текущий план.
  • Удалить задание ... - удаляет выбранное задание из текущего плана.
  • Переместить задание вверх - перемещает выбранное задание на одну позицию вверх в текущем плане.
  • Переместить задание вниз - перемещает выбранное задание на одну позицию вниз в текущем плане.
  • Добавить тест ... - запускает диалоговое окно Добавить тест для добавления теста к текущему заданию.
  • Удалить тест ... - удаляет выбранный тест из текущего задания.

Дополнительная вкладка "План"

Графическое представление плана, которое также показывает состояние плана при его выполнении.
Вы можете редактировать любой из элементов плана, дважды щелкнув по ним.

Дополнительная вкладка вывода

Любой вывод, созданный при загрузке или запуске плана.

Планы на будущее

В будущих версиях этого дополнения будут добавлены:
  • Поддержка нескольких планов
  • Создание плана из текущей конфигурации
