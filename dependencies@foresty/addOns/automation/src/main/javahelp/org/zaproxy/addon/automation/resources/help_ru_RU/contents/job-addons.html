Automation Framework - работа над надстройками

Это задание позволяет вам управлять надстройками ZAP.

Идентификаторы надстроек для использования в списках «установка» и «удаление» отображаются в диалоговом окне «Управление надстройками» ZAP Desktop или в диалоговом окне «Управление надстройками». можно найти на странице ZAP Marketplace .

YAML

  - type: addOns                       # Управление дополнениями 
    parameters:
      updateAddOns: false              # В настоящее время отключен из-за проблем с обновлением фреймворка и заданий во время их работы. 
    install:                           # Список нестандартных надстроек для установки с ZAP Marketplace 
    uninstall:                         # Список стандартных надстроек для удаления
